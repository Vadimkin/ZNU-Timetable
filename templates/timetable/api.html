Документація

На базі даного API було розроблено додаток на Android.

GET /timetable/

Повертає розклад групи або викладача

Параметри

group ID групи
не обов’язковий параметр
teacher ID викладача
не обов’язковий параметр
periodicity Періодичність:
0 — завжди
1 — чисельник
2 — знаменник
не обов’язковий параметр

Результат

date_start День початку семестру
date_end День кінця семестру
periodicity Періодичність:
0 — завжди
1 — чисельник
2 — знаменник
day День тижня:
0 — понеділок
1 — вівторок
...
6 — неділя
lesson_type Тип предмета:
0 — не задано
1 — лекція
2 — семінар
3 — лабораторна робота
4 — консультація
5 — екзамен

Приклад запиту:

GET http://{{request.META.HTTP_HOST}}/api/v1/timetable/?group=1&format=json

GET /department/

Повертає список всіх факультетів університету та timestamp останнього оновлення

Приклад запиту:

GET http://{{request.META.HTTP_HOST}}/api/v1/department/?format=json

GET /group/

Повертає список всіх груп, факультет кожної та timestamp останнього оновлення

Параметри

id ID групи
не обов’язковий параметр
department ID факультета
не обов’язковий параметр

Приклад запиту:

GET http://{{request.META.HTTP_HOST}}/api/v1/group/?format=json

GET /teacher/

Повертає список всіх викладачів

Параметри

id ID викладача
не обов’язковий параметр
department ID факультета
не обов’язковий параметр

Приклад запиту:

GET http://{{request.META.HTTP_HOST}}/api/v1/teacher/?department=1&format=json

Приклад запиту:

2 – знаменник

Приклад запиту:
